## 📌 About

**School Attendance Register** is a fully production-ready web application that allows schools to manage daily student attendance digitally. It features a role-based login system for Admins and Teachers, real-time attendance tracking, and exportable reports — all from a clean, mobile-friendly interface.

## ✨ Features

### 🔐 Authentication & Role-Based Access
- Secure login with username & password
- **Admin** — Full control over classes, students, teachers & reports
- **Teacher** — Access to assigned class only for marking attendance
- Form validation with clear error messages

### 🛠️ Admin Dashboard
- **Classes & Students Tab** — Add/Edit/Delete classes and students, expandable class cards with student table
- **Teachers Tab** — Add teachers with photo, name, subject, assigned class, username & password; Edit/Delete with confirmation toasts
- **Reports Tab** — Monthly attendance report per class, daily columns for each student, total Present/Absent count, Export to **PDF** or **CSV**, Clear Data option

### 📋 Teacher Dashboard
- Date picker (defaults to today's date)
- Live summary: Total Students, Present, Absent
- Toggle each student Present / Absent individually
- **Mark All Present** button
- Save attendance with loading spinner
- Message shown if no class is assigned

### 👤 Profile & Settings
- Edit profile (name, photo)
- Set/update password (Admin)
- Contact Developer option in menu
- Logout

## 📖 How to Use

**As Admin:**
1. Log in with admin credentials
2. Go to **Classes & Students** tab → Add your classes and students
3. Go to **Teachers** tab → Add teachers and assign them a class
4. Go to **Reports** tab → View monthly attendance, export PDF/CSV

**As Teacher:**
1. Log in with teacher credentials (provided by admin)
2. Select date (defaults to today)
3. Toggle each student Present/Absent or use **Mark All Present**
4. Click **Save Attendance**
